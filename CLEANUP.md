# Файлы которые можно удалить

- database.js (дубликат серверного файла)
- server.js (дубликат серверного файла)
- .env (дубликат серверного файла)
- package.json (дубликат, есть в /server)

- package-lock.json (генерируется автоматически)

## В папке deploy/:
- .git/ (встроенный репозиторий не нужен)
- server/ (дубликат)

## Общие:
- .vscode/ (настройки редактора)
- start.bat (заменён на RUN.bat)
